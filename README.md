# 🖱️ AI Virtual Mouse using Hand Gesture Recognition

A real-time computer vision project that allows users to control mouse movement and clicks using hand gestures captured through a webcam.

---

## 🚀 Features
- Control mouse cursor using hand movements
- Left click using finger gestures
- Right click using gesture detection
- Smooth and stable cursor movement

---

## 🧠 How It Works
- Uses **MediaPipe** to detect 21 hand landmarks
- Tracks finger positions in real-time
- Maps hand coordinates to screen resolution
- Detects gestures based on finger positions and distances
- Performs system actions like mouse movement and clicks

---

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- AutoPy
